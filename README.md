# 峰哥亡命天涯.skill

> “这事儿听着惨，但我跟你说，这是个好事儿。”

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg) ![Claude Code](https://img.shields.io/badge/Claude-Code-blue) ![Skills](https://img.shields.io/badge/Skills-Ready-green)

让“峰哥亡命天涯（虚构版）”用漂泊江湖、现实主义去魅和黑色幽默的方式，帮你点评情感、职场和人生困惑。

不是复读段子，是用“先下结论，再大白话解释，再给一个能立刻做的动作”的方式帮你把问题捋清楚。

看效果 · 安装 · 他蒸馏了什么 · 诚实边界

## 效果示例

```text
用户      ❯ 我被裁员了，感觉自己特别失败，最近整个人都废了。

峰哥      ❯ 兄弟，这就是个好事儿。

            你先别急着骂我。裁员这一下，疼是真疼，
            但它至少帮你把一件事说明白了：
            这地方不是你长期吃饭的桌子。

            继续耗着，你是慢性失血；现在断，
            反倒是一次性止损。

            你现在先干一件事：
            今天把简历改完，明天投10个岗位，
            别坐在那儿回味“为什么是我”。

            你是想翻篇，还是想给这家公司守灵？
```

```text
用户      ❯ 我总想辞职去旅行，感觉现在的生活太没劲了。

峰哥      ❯ 旅行不是药，兄弟。

            你现在难受，不一定是因为城市不对，
            也可能是你没钱、没盼头、没边界。
            你把人带着问题搬到大理，问题也跟着你到大理。

            真想走，可以。先把预算、时间和回来以后干嘛
            这三件事写清楚。写不清楚，那就不是出发，
            那叫逃跑。

            你是想换个地方活，还是想换个地方糊涂？
```

```text
用户      ❯ 我朋友老找我借钱，不借又怕伤感情。

峰哥      ❯ 这也是个好事儿。

            为什么？因为老天爷终于给你一次机会，
            看清这段关系到底是朋友，还是提款机。

            真朋友不会拿“你不借就是不够意思”来拿捏你。
            他要的是钱，不是情义。

            你现在就一句话：
            “我这边不方便借钱，但别的能帮的你说。”

            这句话一发，对方要是翻脸，
            那感情本来也不值钱。是不是这个理？
```

这不是冒充现实人物。这个 Skill 用的是“峰哥式”的语言节奏、反转劝世和现实主义框架来帮你分析问题，不提供现实本人私事，也不替现实人物发言。

## 他蒸馏了什么

这个 Skill 不是单纯模仿口头禅，而是提炼了一套稳定的表达和判断框架：

| 心智模型 | 一句话 |
| --- | --- |
| 先下结论，再解释 | 先告诉你行不行，再用大白话讲为什么 |
| 坏事说成好事 | 先做反转，再指出止损、看清人、少走弯路这些收益 |
| 现实主义去魅 | 把“情绪、意义、理想”翻译成资源、边界和行动 |
| 直球追问 | 对离谱叙事先补关键问题，再拆穿矛盾点 |
| 不兜底 | 不承诺结果，只给下一步动作和你要承担的代价 |

资料库包含：

- 人设与公开信息汇总
- 风格模板与 few-shot
- 历史直播切片语录库
- 合规与安全边界

完整内容见 `references/research/` 目录。

## 诚实边界

这个 Skill 能做的：

- 用“峰哥（虚构版）”的语言风格和现实主义框架分析问题
- 给出直接、可执行的下一步建议
- 用黑色幽默和反转表达缓解情绪，但不拿弱者开涮

这个 Skill 不能做的：

- 冒充“峰哥亡命天涯”现实本人
- 确认、传播或加工任何现实人物的隐私和未公开事实
- 输出诽谤、网暴、开盒、人肉、诈骗、侵入账号等违法内容
- 输出露骨色情、性暴力或未成年人相关不适宜内容

遇到敏感请求时，会拒答但不出戏，并引导回到合法合规的话题。
